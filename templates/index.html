How To Get App Password

  1. Open Gmail, then click your profile photo and choose Manage your Google Account.
  2. Open the Security tab and make sure 2-Step Verification is enabled.
  3. Search for App Passwords from the Google Account security area.
  4. Generate the 16-character password and paste it into this form.
  5. Preview first, then send the batch when the counts and sample emails look correct.

Viru Auto Mail Sender

Everything you need to send bulk Gmail emails from one page, without storing your data.

This frontend is designed for a simple workflow: upload email.csv, preview the rendered emails, attach your resume if needed, enable test mode when you want a small batch, and then send the campaign while the terminal shows live progress.

How It Works

  • The app accepts only email.csv and checks the file before sending.
  • Preview mode renders the first three emails so you can verify placeholders and formatting.
  • Duplicate and invalid email addresses are skipped automatically before delivery.
  • Test mode limits the run to the first valid recipients so you can verify the flow safely.
  • An optional resume file can be attached to every email for the current session only.
  • No database is used, so form data and files are handled in memory during the active job only.

Placeholder Support

  • Use {% raw %}{{name}}{% endraw %}, {% raw %}{{company}}{% endraw %}, {% raw %}{{email}}{% endraw %}, {% raw %}{{city}}{% endraw %}, and {% raw %}{{job_role}}{% endraw %}.
  • You can also use exact CSV header names such as {% raw %}{{Company_Name}}{% endraw %} or {% raw %}{{Source_Link}}{% endraw %}.
  • The live counter cards show total rows, queued recipients, sent emails, failed emails, invalid emails, and duplicates.
  • The delivery terminal streams status lines like Sent to: careers@example.com in real time.
